LOL_AVENTURE


Plateforme utilisé :
Windows 7

Logiciel de programmation :
Eclipse

Langage de programmation :
Java

Ce projet a été réalisé lors de la période des grandes vacances après mon année d'intégration à l'ENSIBS.
Ce projet a été fait par LUCK Bastien.

Le jeu est loin d'être fini (seul le setup de départ existe réellement) mais ne sera pas modifié donc il est considéré en version finale.
Contrairement aux autres jeux, la raison de la non modification de ce programme n'est pas que je considère le jeu suffisamment jouable pour ne pas avoir réellement à le toucher et que mieux vaut voir une autre mécanique de gameplay mais plutôt que le programme est juste trop lourd (pour la lecture ainsi que l'ordinateur (/!\ à ne pas le faire tourner trop longtemps car il utilise fortement le CPU (50-75% sur mon viel ordi))).
Il est possible qu'un jour je retourne sur la programmation de ce jeu qui avait pour but d'être un hub à plein d'autres jeux (au moins 1 par personnage était prévu (pas tous à la sortie mais avec le temps... 10-20 ans après le début du projet si LOL serait toujours vivant celà aurait pu faire un bon spin off qui sait ? :) )) mais je repartirait from scratch pour mieux faire la structure du code qui est actuellement beaucoup trop calculatoire pour l'ordinateur (et peu pratique à écrire, il faut l'admettre xd).

Il n'y a pas de rapport pour ce jeu car il était un projet purement personnel.
Le jeu avait pour présemption d'être adaptablle à toute taille d'écran et de proposer un grand panel de jeux différents (certains très simple comme un pierre/feuille/ciseaux pour twisted fate par exemple), avec pour certains un menu option dédié.
Le jeu se joue avec la souris pour le menu option et avec le clavier pour le déplacement (flèche haut/bas) sur le menu principal.

aucune documentation existe car je ne connaissais pas le concept à l'époque.

Pour lancer le programme :
ouvrir une console puis
cd path/to/READ/ME
java -jar LOL_AVENTURE.jar
